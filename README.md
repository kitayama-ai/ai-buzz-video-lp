# AIバズ動画 LP - デザイン改善版

## フォルダ構成

このLPは独立したフォルダ `ai-buzz-video-lp` に配置されています。
他のLPプロジェクト（`female-lp`、`video-editor-lp`など）とは完全に分離されており、上書きのリスクはありません。

## ファイル構成

```
ai-buzz-video-lp/
├── index.html      # HTML構造
├── styles.css      # スタイルシート
├── script.js       # JavaScript（アニメーション制御）
└── README.md       # このファイル
```

## 主な改善点

### デザイン改善
- **余白の最適化**: テキストの密集を解消し、読みやすさを向上
- **タイポグラフィ**: Noto Sans JPを使用し、視認性と高級感を両立
- **カラースキーム**: ダークトーンを維持しつつ、より洗練されたグラデーション
- **視覚的階層**: 明確な情報の優先順位付け

### インパクト強化
- **ヒーローセクション**: グラデーション背景とアニメーション効果
- **CTAボタン**: ホバーエフェクトとパルスアニメーション
- **コンテンツブロック**: ホバー時のエレベーション効果
- **スクロールアニメーション**: Intersection Observer APIを使用

## 使用方法

1. `index.html`をブラウザで開く
2. または、ローカルサーバーで実行

## 注意事項

- 構成やテキスト内容、トンマナは既存LPを維持
- デザイン性のみを大幅にブラッシュアップ
- 他のLPプロジェクトとは完全に独立したフォルダ構成
